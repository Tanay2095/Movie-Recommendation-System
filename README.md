# 🎬 Movie Recommendation System

A content-based movie recommendation system built using Python, Pandas, Scikit-Learn, and Streamlit.

## 📌 Project Overview

This application recommends movies similar to a selected movie based on genre similarity.

The recommendation engine uses:

- CountVectorizer
- Cosine Similarity

to find movies with similar genres and suggest them to the user.

---

## 🚀 Features

- Interactive Streamlit web interface
- Movie selection using dropdown menu
- Top 5 similar movie recommendations
- Error handling for invalid inputs
- Fast recommendation generation

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

---

## 🧠 Machine Learning Concepts

### Feature Extraction

CountVectorizer converts movie genres into numerical vectors.

### Similarity Measurement

Cosine Similarity is used to calculate how similar two movies are.

---

## 📂 Dataset

Dataset used:

MovieLens Small Dataset

Contains:

- 9,742 movies
- Movie genres
- User ratings

---

## ▶️ Run Locally

Install dependencies:

```bash
pip install pandas numpy scikit-learn streamlit
```

Run the application:

```bash
streamlit run app.py
```
